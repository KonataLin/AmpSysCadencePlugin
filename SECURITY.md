# Security Policy

AmpSys Cadence Plugin ships public GUI/SKILL wrappers plus protected binary core packages.

## Safety Notice

AmpSys Cadence Plugin is an early-design aid for sizing exploration and workflow validation. It is not a replacement for complete simulation, layout verification, PVT/Monte Carlo/reliability checks, or tape-out sign-off.

This plugin does not include any PDK, process model, foundry file, simulator, or commercial EDA license. Users must legally provide and configure their own process libraries and tools.

AmpSys Cadence Plugin is provided as a free public-interest project. The author is not responsible for tape-out failure, financial loss, schedule loss, design errors, or other consequences caused by use, local modification, redistribution, or misuse of this plugin.

## Reporting

Please report security-sensitive issues privately when possible, or open a GitHub issue with sensitive paths and proprietary PDK data removed.

Do not attach:

- proprietary PDK model files
- company schematics
- internal AmpSys source packages
- private license files

## Supported Builds

The latest GitHub prerelease is the supported public test build unless otherwise noted in the release page.
